# C Programming Projects

This repository contains a collection of C programming projects covering various topics. Each project is designed to help you learn and practice different aspects of C programming.

| Project | Description |
|---------|-------------|
| [0x00-hello_world](./0x00-hello_world) | This project introduces the basics of C programming by focusing on writing simple "Hello, World!" programs. It covers fundamental concepts such as syntax, compiling, and running C code, providing a solid foundation for further exploration of the language. |
| [0x01-variables_if_else_while](./0x01-variables_if_else_while) | This project focuses on using variables, conditional statements, and loops in C. It includes exercises that teach how to control program flow with if, else, and while statements, helping you build foundational programming skills essential for creating more complex C programs. Through hands-on tasks, you'll learn to manipulate data and make decisions in your code. |
| [0x02-functions_nested_loops](./0x02-functions_nested_loops) | This project introduces the fundamental concepts of functions and nested loops in C programming. You'll learn to define functions to perform specific tasks and use loops within those functions to handle repetitive actions. Key exercises include printing the alphabet, calculating values, and generating multiplication tables. By the end, you'll gain a solid understanding of how to structure your code using functions and control flow, enhancing both readability and maintainability.  |
| [0x03-debugging](./0x03-debugging) | This project focuses on debugging techniques in C. It involves identifying and fixing errors in code to ensure that programs run correctly. You will learn how to use debugging tools and strategies, including using print statements and debuggers, to track down and resolve issues. This will help improve your problem-solving skills and understanding of code behavior. |
| [0x04-more_functions_nested_loops](./0x04-more_functions_nested_loops) | This project builds on the concepts of functions and nested loops in C programming. It involves creating various functions that handle tasks like checking character types, printing patterns, and solving common problems using loops. You will practice implementing functions for basic arithmetic operations, number printing, and shape drawing, reinforcing your understanding of how to structure code effectively while utilizing loops and function calls. |
| [0x05-pointers_arrays_strings](./0x05-pointers_arrays_strings) | This project focuses on the use of pointers, arrays, and strings in C programming. It includes functions that manipulate string data, swap values using pointers, reverse strings, and print specific characters from strings. By completing these tasks, you'll gain a deeper understanding of how to manage memory effectively and work with different data structures in C, reinforcing your knowledge of fundamental programming concepts. |
| [0x06-pointers_arrays_strings](./0x06-pointers_arrays_strings) | This project focuses on advanced string manipulation using pointers and arrays in C. It includes functions for string concatenation, comparison, and modification, such as converting characters to uppercase and encoding strings. Additionally, it covers tasks like reversing arrays and printing buffers in a formatted manner. By completing these exercises, you'll strengthen your understanding of memory management and string handling in C, crucial skills for effective low-level programming. |
| [0x07-pointers_arrays_strings](./0x07-pointers_arrays_strings) | This project explores advanced manipulation of pointers, arrays, and strings in C. It includes implementations of common functions for memory operations, string handling, and character searches. Through these exercises, you will gain practical experience in using pointers to manage memory efficiently and perform operations on strings and arrays. Key functions cover memory setting, copying, substring searching, and printing data structures like chessboards and matrices, enhancing your understanding of C's capabilities in low-level programming. |
| [0x08-recursion](./0x08-recursion) | This project delves into the concept of recursion in C programming. It covers various tasks that demonstrate how recursion can be used to solve problems, such as printing strings, calculating factorials, and checking for prime numbers. Each function is designed to highlight the power of recursion in breaking down complex problems into simpler sub-problems, enhancing your understanding of function calls and stack memory. The project includes functions for string manipulation, mathematical computations, and pattern matching, providing a comprehensive look at recursive techniques in C. |
| [0x09-static_libraries](./0x09-static_libraries) | This project focuses on the creation and utilization of static libraries in C programming. It includes the development of a static library named libmy.a that contains a variety of functions, such as string manipulation, character checks, and memory operations. Additionally, the project provides a script (create_static_lib.sh) to automate the process of building a static library, reinforcing the importance of static libraries in organizing and reusing code efficiently. By compiling and linking these libraries, you can streamline your development process and enhance code modularity. |
| [0x0A-argc_argv](./0x0A-argc_argv) | This project focuses on handling command-line arguments in C using the argc (argument count) and argv (argument vector) parameters. It includes several tasks that demonstrate how to access and manipulate these arguments. The tasks range from printing the program's name to performing operations like multiplication and addition on the provided arguments. Additionally, the project covers practical applications, such as calculating the minimum number of coins needed for a given amount of change, showcasing the versatility of command-line argument processing in C programming. |
| [0x0B-malloc_free](./0x0B-malloc_free) | This project focuses on dynamic memory management in C, utilizing malloc and free for memory allocation and deallocation. It includes several tasks that illustrate various applications of dynamic memory, such as creating arrays, duplicating strings, and handling multi-dimensional arrays. The project emphasizes safe memory practices, ensuring proper allocation checks and freeing of memory to prevent leaks. Additionally, advanced tasks involve concatenating program arguments and splitting strings into words, showcasing the flexibility of dynamic memory in C programming. |
| [0x0C-more_malloc_free](./0x0C-more_malloc_free) | This project expands on memory management in C, specifically using malloc, free, and related functions. It includes tasks that demonstrate safe memory allocation with error handling, string concatenation, zero-initialized memory allocation with calloc, and creating arrays with specified ranges. Additionally, it covers reallocating memory and performing operations like multiplying positive numbers. The focus is on effective memory management practices to prevent leaks and ensure proper usage. |
| [0x0D-preprocessor](./0x0D-preprocessor) | This project explores the use of preprocessor directives and macros in C programming. It includes defining object-like macros, mathematical constants like PI, and utilizing function-like macros for operations such as calculating absolute values and sums. The tasks emphasize how preprocessor macros can enhance code readability and maintainability while allowing for easier code modifications. Additionally, it demonstrates how to retrieve the name of the compiled file using a macro. |
| [0x0E-structures_typedef](./0x0E-structures_typedef) | This project focuses on the use of structures and the typedef keyword in C programming. It includes defining a struct dog with specific attributes, initializing and printing instances of this structure, and creating an alias dog_t for the struct dog. Additionally, it covers memory management by allocating and freeing memory for dog structures. The tasks illustrate how to create and manage custom data types effectively in C. |
| [0x0F-function_pointers](./0x0F-function_pointers) | This project explores the concept of function pointers in C programming. It includes various tasks that demonstrate how to use function pointers to enhance flexibility and modularity in code. The tasks involve printing names using function pointers, executing a function on each element of an array, searching for integers within arrays, and performing arithmetic operations through function pointers. Additionally, it provides an example of printing the opcodes of the main function, showcasing the low-level aspect of function pointers. Overall, this project emphasizes the power of function pointers for creating more dynamic and adaptable C programs. |
| [0x10-variadic_functions](./0x10-variadic_functions) | This project focuses on variadic functions in C, which allow functions to accept a variable number of arguments. It includes tasks that demonstrate various applications of variadic functions, such as calculating the sum of parameters, printing numbers and strings with optional separators, and printing different data types based on a specified format. These tasks illustrate the flexibility of variadic functions, enabling more dynamic and reusable code. The project emphasizes the importance of handling varying argument types and counts in function definitions. |
| [0x11-printf](./0x11-printf) | Implement a custom printf function in C that replicates the behavior of the standard library printf, focusing on format specifiers and efficient buffer usage. |
| [0x12-singly_linked_lists](./0x12-singly_linked_lists) | This repository contains C programs that demonstrate operations on singly linked lists, including printing elements, counting nodes, adding nodes at both the beginning and end, and freeing the list. It also includes a unique assembly program that prints a specific message. Each program is designed to illustrate different aspects of linked list management and can be compiled and executed using GCC and NASM. |
| [0x13-more_singly_linked_lists](./0x13-more_singly_linked_lists) | This directory contains C programs focused on advanced operations for singly linked lists, including functions for printing, counting, adding, deleting, and freeing nodes. The tasks also cover more complex functionalities such as reversing lists, handling loops safely, and finding loops within linked lists. Each program is designed to enhance understanding of linked list data structures in C and demonstrates a variety of linked list manipulations through clear prototypes and source code links. |
| [0x14-bit_manipulation](./0x14-bit_manipulation) |  | This repository contains C programming projects focused on bit manipulation and endianness. It includes functions for converting binary strings to unsigned integers, printing binary representations without using arrays, and manipulating individual bits through various operations like setting, clearing, and flipping bits. Additionally, it provides a method for checking the endianness of the system and a task for cracking a password. Each project emphasizes practical applications of bitwise operations in C programming. |
| [0x15-file_io](./0x15-file_io) | This repository features C programs dedicated to various file I/O operations. It includes tasks for reading text files, creating new files with specified permissions, appending text to existing files, copying file contents with error handling, and displaying ELF header information from ELF files. Each program is designed to illustrate specific file-handling capabilities in C, and users can compile and execute them using the provided instructions for testing file operations. |
| [0x16-simple_shell](./0x16-simple_shell) | This project involves creating a simple Unix shell that executes commands, manages input and output, and supports features like piping and redirection. Users will gain insights into how shells operate and learn to implement functionalities such as built-in commands, variable expansion, and logical operators, providing a foundational understanding of command-line interfaces in Unix-like systems. |
